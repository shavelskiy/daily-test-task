# Daily test task frontend

### Стэк:

- Nextjs 13
- Nodejs 16.14
- Bootstrap

### Команды Make

| Команда | Описание                      |
| ------- | ----------------------------- |
| start   | Запускает проект в docker     |
| stop    | Отсанавливает проект          |
| install | Устанавливает зависимости     |
| lint    | Запуск линтера                |
| dev     | Запуск локального dev сервера |
